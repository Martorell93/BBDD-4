Información Notas

    • Leyenda:

    • Asignatura: Id de la asignatura en la BBDD
    • Profesor: Id del profesor en la BBDD
    • Alumno: Id del alumno en la BBDD
    • Fecha: Fecha de la realización del test
    • Nota: Nota obtenida en el test
    • Id alumno: Id de alumno en la BBDD
    • Id nota: Id de nota en la BBDD

    Botones:

    • Mostrar: Muestra información sobre las notas de X alumno, identificando el id alumno
    • Crear: Crea una nota con los campos rellenos,excepto el id nota
    • Actualizar: Identificar id nota, se modifica la información de la nota
    • Eliminar: Poniendo el id, se elimina la información de esa nota
